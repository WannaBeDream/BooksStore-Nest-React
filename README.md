## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

Go to http://localhost:3000/swagger to check more information.

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
